# Haldiram Customer Satisfaction Survey (SQL Project)

## Project Overview
This project analyzes customer satisfaction data for Haldiram using SQL.

## Dataset
- 500 customer survey responses
- Fields: age, gender, visit frequency, product quality, price satisfaction, staff behavior, overall rating

## Skills Used
- SQL
- Data Analysis
- Customer Insights

## Use Case
This project helps understand customer preferences and improve service quality.
